== Description ==

The Signature Widget for Genesis adds a signature widget on single posts. Bloggers are able to add their signature images along with any other item they wish to have displayed with their signature.

You can see the demo [here](http://cre8tivediva.com/downloads/genesis-signature-widget/)

== Installation ==

Via the Dashboard

After you have downloaded your product:

*	Go to Plugins
*	Select Add New
*	Search for Signature Widget for Genesis
*	Install
*	Activate the plugin through the Plugins menu in WordPress
*	Go to Appearance -> Widgets and you will see the Signature Widget.
*	Install the Image Widget from Jetpack, Modern Tribe or similar.
*	Create a signature image.
*	Drag the Image Widget over to the Signature Widget for Genesis area.
*	Upload your signature image and save.

Via FTP

*	Download and unzip the Signature Widget for Genesis zip file from the WordPress Repository.
*	Via FTP, upload the entire `genesis-signature-widget` folder to your `/wp-content/plugins` directory.
*	Activate the plugin through the Plugins menu in WordPress.
*	Go to Appearance -> Widgets and you will see the Signature Widget.
*	Install the Image Widget from Jetpack, Modern Tribe or similar.
*	Create a signature image.
*	Drag the Image Widget over to the Signature Widget for Genesis area.
*	Upload your signature image and save.


== Frequently Asked Questions ==

= Do I need to have the Genesis Framework installed? =

Not since 1.3.0.

= What is the recommended image size? =

If you want to avoid responsive issues, use an image no more than 320px wide.
